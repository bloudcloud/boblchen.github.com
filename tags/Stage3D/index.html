单位矩阵

有一种特殊的矩阵称为单位矩阵(identity matrix)。单位矩阵是一个正方形矩阵,它除了对角线上的元素为1外,其他元素均为0。如下图分别为2x2、3x3、4x4单位矩阵:


结合到之前谈到的矩阵乘法定义。因为单位矩阵只有对角线为1，其它全为0。那么如果一个单位矩阵乘以其它矩阵，或者其他矩阵乘以一个单位矩阵时，结果仍然是该矩阵。如果A是一个𝑚×𝑛矩阵,B是一个𝑛×𝑝矩阵,I是一个𝑛×𝑛单位矩阵。那么AI=A。IB=B。

## 逆矩阵
矩阵代数中没有除法运算,但是它定义了一种乘法的逆(inverse)运算。逆矩阵有如下几个特点:
只有𝑛×𝑛矩阵可以做逆运算。 一个𝑛×𝑛的逆矩阵仍然是一个𝑛×𝑛矩阵。
不是所有的𝑛×𝑛矩阵都有逆矩阵 如果存在逆矩阵，那么这个逆矩阵是唯一的。
* 将一个矩阵与它的逆矩阵相乘，那么结果必定为单位矩阵。
逆矩阵在求解矩阵时，非常有用。例如:一个向量矩阵A乘以矩阵B得到向量矩阵C。那么这个时候就可以通过向量矩阵C乘以矩阵B逆矩阵得到向量矩阵A。

stage3d 矩阵

在stage3d中使用了两种方式的矩阵，一种是正常的，另外一种是转置了的矩阵。在cpu层面,as3提供的api是正常矩阵，也是D3DX使用的矩阵。但是在gpu层面使用的经过转置的矩阵，也就是opengl使用矩阵。它们不同之处在于运算的时候，前者是右乘，后者则是左乘。

关于向量和矩阵

在3D编程中，向量和矩阵是最基础的只是。因为被大量用到。因此谈到3D时需要先谈一下向量和矩阵的概念以及向量和矩阵之间的关系。后面就会讲到向量以及矩阵在3D中运用。

向量

在数学中，向量指具有大小和方向的几何对象，可以形象化地表示为带箭头的线段：箭头所指，代表向量的方向；线段长度，代表向量的大小。一个向量可以有多种记法，如记作粗体的字母（a、b、u、v），或在字母顶上加一小箭头→，或在字母下加波浪线~。如果给定向量的起点（A）和终点（B），可将向量记作AB（并于顶上加→）。

几何表示

向量可以用有向线段来表示。有向线段的长度表示向量的大小，向量的大小，也就是向量的长度。长度为0的向量叫做零向量.长度等于1个单位的向量，叫做单位向量。箭头所指的方向表示向量的方向。（若规定线段AB的端点A为起点，B为终点，则线段就具有了从起点A到终点B的方向和长度。

坐标表示

在平面直角坐标系中，设点A=(x1,y1)，点B=(x2,y2)。则向量AB=(x2-x1,y2-y1)。对于多维的空间向量，通过类推得到。向量的模
向量的大小，也就是向量的长度（称为模）。向量a的模记作|a|。向量的模是非负实数。设向量a=(x,y)，|a|=math.sqrt(xx+yy)。

单位向量

模为1的向量，称之为单位向量。某些情况下面，我们为了计算方便，会将模不为1的向量转换为模为1的向量。例如：玩家角色从A点移动到B点。那么只需要计算出A到B的向量，将向量AB转换为模为1，让玩家从当前位置朝着方向AB移动”模 * speed”距离即可。计算方法：设向量AB=(x,y)。则模为1的向量=(x/|a|, y/|a|)。|a|表示向量的模。

向量点积

设向量a=(x1,y1)，向量b=(x2,y2)。则a·b=x1x2 + y1y2。如果两个向量均未单位向量，那么两个向量的点积就得到两个向量的夹角的cos值。如果cos值越大，那么夹角就越小,cos值越小，夹角就越大。例如：a·b=1时，夹角为0，a·b=0时，夹角为90°。利用单位向量的点击可以用来模拟光照等效果。假设一个三维空间，X轴向右，Y轴向上，Z轴向前。一个平面平铺在xz轴上，用一个向量来表示该平面的方向：a=(0, 1, 0)。假设一个灯光方向为(0, -1, 0)。该灯光垂直照向于该平面。因为屏幕是朝上，但是灯光是照向平面，因此灯光的方向永远都是跟平面的方向相反，因此为了计算方便，我们将灯光方向取反->(0, 1, 0)。则a·b=1。夹角为0，说明灯光垂直照向平面，那么这个时候应该是平面受光最多，平面最亮。以256色为例，设平面颜色为rbg(0, 0, 0)黑色。灯光颜色(255, 0, 0)红色。那么我们可以使用最简单的加法法则：(0, 0, 0) + (255, 0, 0) * 1.0 = (255, 0, 0)。这个时候平面就会表现出红光。如果灯光方向为(1, 0, 0)。说明灯光方向与屏幕互相平行。平面是不会受到光照的。经过计算得出：a·b=0。最终屏幕rgb(0, 0, 0)仍然为黑色。

向量叉积

两个向量a和b的叉积axb的结果为c，c为一个新向量，c垂直于a和b。在3d中，向量叉积使用颇多。例如：前一节向量点积中，我们定义屏幕的方向为(0, 1, 0)。如果我们只知道平面其中的三个不重叠的点以及灯光方向，这个时候需要我们计算出平面受到的光照。设三个点分别为A,B,C。那么向量AB=B-A，向量AC=(C-A)。然后根据向量ABxAC即可计算出一条垂直于该平面的向量，那么这条向量就可以用于光照的计算。
